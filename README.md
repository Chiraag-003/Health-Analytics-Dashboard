Health & Wellness Risk Analytics Dashboard

📊 Overview
This project analyzes fitness and health data to identify risk levels and understand factors affecting recovery and wellness.

🚀 Objectives
- Identify high-risk individuals
- Analyze impact of sleep, stress, and activity
- Build actionable insights

🛠 Tools Used
- Python (Pandas, Matplotlib)
- Power BI

📈 Key Features
- Custom metrics:
  - Wellness Score
  - Recovery Score
- Risk segmentation:
  - High Risk
  - Medium Risk
  - Low Risk
- Interactive dashboard

📊 Key Insights
- Users sleeping 7–8 hours show best recovery
- High stress significantly reduces recovery
- Low-risk users sleep ~2 hours more than high-risk users
